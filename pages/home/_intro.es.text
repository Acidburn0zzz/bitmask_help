*Bitmask* es una aplicación de código abierto para hacer las comunicaciones cifradas más fáciles y seguras. Puedes elegir entre [[diferentes proveedores de servicios => #providers]] o [[montarte el tuyo propio => https://leap.se/es/doc/platform]]. Actualmente Bitmask soporta Internet Cifrada (VPN) y Mail Cifrado (con otros servicios en la recámara).
